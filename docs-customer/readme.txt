Building and running the project
================================
* Make sure that you have NodeJS installed.
* Make sure that you have NPM installed.
* Download all the project dependencies with the following command:
npm install
* Run to project with the following command:
npm start
* Node package manager [npm] should download all the required dependencies, start the webserver and open a browser window. Apart from having a working NodeJS and npm install, no other software should be needed to run the web app.
